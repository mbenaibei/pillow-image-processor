# 🖼️ Pillow Image Processor

A configurable image processing utility using Python Pillow library.

## Features

- 🖼️ Batch process images from a directory
- 📐 Resize images with thumbnail generation
- 🎨 Apply image filters (DETAIL)
- ⚫ Convert images to grayscale
- 📁 Supports JPEG, PNG, and GIF formats

## Installation

```bash
# Clone the repository
git clone git@github.com:mbenaibei/pillow-image-processor.git
cd pillow-image-processor

# Install with uv
uv pip install -e .

# Or install with development dependencies
uv pip install -e .[dev]
```
